# X UI Clone

A responsive, single-page X (Twitter) style clone implemented with plain HTML, CSS, and JavaScript.

## Included features

- Responsive 3-column layout (sidebar, timeline, widgets)
- For You / Following timeline tabs
- Post composer with 280-char counter and optional image URL
- Persistent timeline data via `localStorage`
- Search filter across author and post text
- Like and repost toggles with live counts
- Dynamic trends extraction from hashtags
- Dark mode toggle with saved preference

## Run locally

Open `index.html` in your browser.
