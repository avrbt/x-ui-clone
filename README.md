# 🚀 X-UI Clone

A modern UI clone project built to practice and demonstrate frontend development skills.
This project focuses on clean design, responsive layouts, and efficient use of HTML, CSS, and JavaScript.

---

## ✨ Features

* 🎨 Clean and modern UI design
* 📱 Fully responsive layout
* ⚡ Fast and lightweight
* 🧩 Organized folder structure
* 🔧 Easy to customize

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**

---

## 📁 Project Structure

```
x-ui-clone/
│── css/
│── js/
│── index.html
│── README.md
```

---

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/avrbtx/x-ui-clone.git
```

2. Open the project

```bash
cd x-ui-clone
```

3. Run in browser
   Just open `index.html` in your browser.
